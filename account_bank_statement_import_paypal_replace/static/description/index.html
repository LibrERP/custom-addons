Import PayPal CSV files as Bank Statements in Odoo / N/D

This module allows you to import the PayPal CSV files in Odoo as bank statements.

N/D

Usage / Utilizzo

To import statements from PayPal via Statement report:

#. Go to PayPal #. Open Statements and select Monthly or Custom #. Select date range of interest and click Request under CSV column #. Wait for statement report to be generated #. Download it and save to a file #. Go to Odoo and and import saved statement file, selecting corresponding format

To import statements from PayPal via Activity report:

#. Go to PayPal #. Open Activity download #. Set Transaction type to Balance affecting #. Set Format to CSV #. Select date range of interest and click Create report #. Wait for activity report to be generated #. Download it and save to a file #. Go to Odoo and and import saved statement file, selecting corresponding format
